Appose u&:v  _ _ _  

&: is equivalent to & except that the ranks of the resulting function are infinite; the relation is similar to that between @: and @ .
 

For example:
   a=: 'abcd' ; 'efgh'
   b=: 'ABCD' ; 'EFGH'
                
   a
+----+----+
|abcd|efgh|
+----+----+

   b
+----+----+
|ABCD|EFGH|
+----+----+

   a ,&:> b
abcd
efgh
ABCD
EFGH

   a ,&> b
abcdABCD
efghEFGH

   > b. 0
0 0 0

   , & > b. 0
0 0 0

   , &: > b. 0
