J 5.01 Non-Compatible Changes initial writing: 2002-05-27
last updated: 2002-08-12

  • 1!:40 (path) no longer works.
  • 1!:41 (server path) no longer works.
  • 1!:42 (library path) no longer works.
  • 2!:4 (command line boxes) no longer works.
  • 2!:6 (command line string) no longer works.
  • 9!:30 and 9!:31 (professional key) no longer work.
  • The domain of 13!:8 (error number) now limited to integers between 1 to 255.
  • 13!:9 (re-run with arguments) no longer works.
  • Foreigns (for example 6!:0) that ignored their arguments previously permitted any argument to be used; now they require an empty vector argument.
  • All tridents (dictionary Section II F) other than the verb-verb-verb case no longer work.
  • All bidents (dictionary Section II F) other than the verb-verb, adverb-adverb, and conjunction with an argument cases no longer work.
  • The conjunction [. (Lev) no longer works.
  • The conjunction ]. (Dex) no longer works.
  • The adverb ]: (Identity) no longer works.
  • The case 11 : y (translate to a tacit adverb) no longer works.
  • The case 12 : y (translate to a tacit conjunction) no longer works.
  • Alternative spellings for national use characters ([ { @ etc.) no longer work.
