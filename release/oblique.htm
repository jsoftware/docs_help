f//. and f//.@(g/) Improved initial writing: 2005-10-18
last updated: 2005-10-28

The monad f//. and the dyad f//.@(g/) are now implemented using special code for certain combinations of datatypes and rank-0 verbs f and g .

+//. for all datatypes; <.//. and >.//. for all but the complex datatype; and f//. for the following f :
Boolean     +. *. = ~: < <: > >:
integer  17 b. 22 b. 23 b.

+//.@(*/) for all datatypes, and:
Boolean     ~://.@(*./)   ~://.@(+./)   +//.@(*./)   +//.@(+./)
integer   22 b.//.@(17 b./)
