colib.ijs

This script defines the class/object library and is included in the J standard library. Definitions are loaded into the z locale.

Definitions

C coclass, cocreate, cocurrent, codestroy, coerase, cofullname, coinsert, coname, conames, conew, conl, copath, coreset

coclass (verb)

Set class/object class.

cocreate (verb)

Create object.

cocurrent (verb)

Set current locale.

codestroy (verb)

Destroy current object.

coerase (verb)

Erase object.

Example

coerase <'jzplot'

cofullname (verb)

Return name with locale qualifier.

coinsert (verb)

adds argument and copath of argument to current path.

paths are in order of argument, except that z is at the end.

Example

coinsert 'cdir'
coinsert 'cdir pobj'
coinsert 'cdir';'pobj'

coname (verb)

Return current class/object name.

conames (verb)

Formatted class/object name list.

conew (verb)

Create object.

conl (verb)

Return class/object name list.

Syntax

conl n

copath (verb)

Set/get class/object path.

coreset (verb)

Destroy object locales,.
