coutil.ijs

This script defines the class/object utilities and is included in the J standard library. Definitions are loaded into the z locale.

Definitions

C cofind, cofindv, coinfo, conouns, conounsx, copathnl, copathnlx, coselect, costate

cofind (verb)

Find locales containing global name y.

cofindv (verb)

Find locales and values for global name y.

coinfo (verb)

Get info on co classes

Returns: noun refs;object;creator;path

conouns (verb)

Nouns referencing object.

conounsx (verb)

Object references with locales returns: object;references;locales

copathnl (verb)

Path name list.

copathnlx (verb)

Formatted path name list with defining classes.

coselect (verb)

Select current locale.

costate (verb)

State of class objects.
