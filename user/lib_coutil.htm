coutil.ijs

This script defines the class/object utilities and is included in the J standard library. Definitions are loaded into the z locale.

Definitions

C cofind, cofindv, coinfo, conouns, conounsx, copathnl, copathnlx, costate
cofind v find locales containing global name y
cofindv v find locales and values for global name y
conouns v nouns referencing object
copathnl v path name list
copathnlx v formatted path name list with defining classes
costate v state of class objects

coinfo (verb)

Get info on co classes

Returns: noun refs;object;creator;path

conounsx (verb)

Object references with locales returns: object;references;locales
