dll.ijs

This script defines utilities for calling DLLs and is included in the J standard library. Definitions are loaded into the z locale.

Definitions

A AND
C cd, cdcb, cder, cderx, cdf
E endian, Endian
F fc, fh
G gh
I ic
J JB01, JBOXED, JCHAR, JCMPX, JFL, JINT, JPTR, JSIZES, JSTR, JTYPES
M mema, memf, memr, memw
O OR
S symdat, symget, symget
X XOR

Nouns

JB01 1
JBOXED 32
JCHAR 2
JCMPX 16
JFL 8
JINT 4
JPTR JINT
JSIZES size in bytes of corresponding JTYPES
JSTR _1,JCHAR
JTYPES JB01,JCHAR,JINT,JPTR,JFL,JCMPX,JBOXED

AND (verb)

Bitwise AND (&).

cd (verb)

Call DLL procedure.

cdcb (verb)

Callback address.

cder (verb)

Error information.

cderx (verb)

GetLastError information.

cdf (verb)

Free DLLs.

endian (verb)

Flip intel bytes (little endian).

Endian (verb)

Flip powerpc bytes (BIG Endian).

fc (verb)

Float conversion.

fh (verb)

Free header.

gh (verb)

Allocate header.

ic (verb)

Integer conversion e.g.

   25185 25699 = _1 ic 'abcd'
   'abcd' = 1 ic _1 ic 'abcd'
   1684234849 1751606885 = _2 ic 'abcdefgh'
   'abcdefgh' = 2 ic _2 ic 'abcdefgh'

mema (verb)

Memory allocate.

memf (verb)

Memory free.

memr (verb)

Memory read.

memw (verb)

Memory write.

OR (verb)

Bitwise OR (|).

symdat (verb)

Get address of data for name.

symget (verb)

Get address of locale entry for name.

symget (verb)

Set array as address.

XOR (verb)

Bitwise XOR (^).
