jal.ijs

This script defines jal utilities and is included in the J standard library. Definitions are loaded into the z locale.

Definitions

G getqtbin
I install

getqtbin (verb)

Get Qt binaries.

Always downloads the jqt binary.

For the qt library (required for mac and win):

y is 0 - download if not present      1 - always download

For linux: y is 'slim' for the slim binaries

For windows: y is 'angle' for the ANGLE binaries

install (verb)

Call pacman install.
