sysenv.ijs

This script defines text system environment and is included in the J standard library. Definitions are loaded into the z locale.

Definitions

F FHS
H hostpathsep
I IF64, IFDEF, IFIOS, IFJCDROID, IFJHS, IFQT, IFRASPI, IFUNIX, IFWIN, IFWINCE, IFWINE, IFWOW64
J jcwdpath, jpathsep, jsystemdefs
U UNAME
W winpathsep

Nouns

FHS filesystem hierarchy: 0=not used 1=linux
IF64 if a 64 bit J system
IFIOS if iOS (iPhone/iPad)
IFJCDROID if JConsole for Android
IFJHS if jhs libraries loaded
IFQT if Qt libraries loaded
IFRASPI if Raspberry Pi
IFUNIX if UNIX
IFWIN if Windows (2000 and up)
IFWINCE if Windows CE
IFWINE if Wine (Wine Is Not an Emulator)
IFWOW64 if running J32 on a 64 bit o/s
UNAME name of UNIX o/s

hostpathsep (verb)

Converts path name to use host path separator.

IFDEF (verb)

If DEFxxx exists.

jcwdpath (verb)

Adds path to J current working directory.

jpathsep (verb)

Converts path name to use / separator.

jsystemdefs (verb)

Loads appropriate netdefs or hostdefs.
