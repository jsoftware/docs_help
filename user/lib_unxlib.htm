unxlib.ijs

This script defines the common shared library for UNIX and is included in the J standard library. Definitions are loaded into the z locale.

Definitions

U UNXLIB, unxlib
UNXLIB n table of common shared libraries

unxlib (verb)

Return the name of a shared library.

Example

unxlib 'c'
