J is a high-level, general purpose programming language. The J system provides: an engine for executing J; various front ends that provide user interfaces to the J engine; a library, written in J, that provides an IDE (interactive development environment), numerous tools, utililties, demos, tutorials; and online documention.

J Systems are available for Windows, OSX, Linux, IOS, Android, Raspberry Pi and others. The core language is identical in all versions, and programs not making use of platform-dependent features will work unchanged on all systems.

J is documented in several online html books that are distributed with the system.

J is licensed for free installation on all platforms. You can download, redistribute, and run end user applications built with J for free.
